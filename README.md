# GitHubSummarizer

## Technologies
    
  - Angular version 1
  - jQuery version 3
  - Google Charts current verions (45 as of October 23, 2016)
  - Bootstrap version 3
 
## Usage
   The project is currently set up to run on a node js server.  The following commands will initialize the project and start the server.
   
```sh
$ cd githubsummarizer
$ npm install
$ npm start
```
